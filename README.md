# Этот код предназначен для родителей абитуриентов, в нем есть кнопки: "Время работы", "Специальность", "Связаться с нами", "Сайт техникума". При нажатии на кнопку "Время работы" показывает время работы техникума. при нажатии на кнопку "Специальность" выводит список всех специальностей и при выборе одной из них показывает стоимотсь обучения и срок обучения. При нажатии на кнопку "Связаться с нами" выводит контактные данные техникума. При нажатии на кнопку "Сайт техникума" выводит кнопку для перехода на оффициальный сайт учебного учереждения.
